📊 Data Science Showcase: Student Performance Analytics 🎓

I just completed an end-to-end data analysis project modeling and visualizing key performance metrics from 600 student academic profiles. The pipeline leverages Python (Pandas) for data engineering and clean quantitative statistics, alongside an interactive high-fidelity glassmorphism frontend dashboard!

🔍 Key Business Questions Solved:
1️⃣ Does attendance affect scores? 
👉 Absolutely. r = +0.81. Attendance below 70% results in a critical drop in averages (35.4%), whilst >=90% attendance drives averages to 78.5%!

2️⃣ Which subjects have the highest failure rates?
👉 Mathematics was the most challenging with a 28.3% failure rate, followed closely by Physics (18.2%). English proved the most accessible (0% failures).

3️⃣ Does study time improve scores?
👉 Solid positive correlation (r = +0.76). Students crossing the 15 hours/week study benchmark hit a 100% pass rate!

💻 Tech Stack Utilized:
✅ Python & Pandas (Correlation, Categorization, Pivot Aggregations)
✅ OpenPyXL (Professional Excel Exporting)
✅ HTML5, CSS3 Glassmorphism, & JavaScript ES6 (Client-side Data Engine)
✅ Chart.js (Responsive Canvas Visualizations)
