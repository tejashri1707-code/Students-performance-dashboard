# 📊 EduMetrics: Student Performance Analytics & Interactive Dashboard

An end-to-end data analytics and visualization showcase. This project utilizes **Python (Pandas)** for statistical analysis, data engineering, and mathematical calculations, alongside a **premium glassmorphism web dashboard (HTML/CSS/JS + Chart.js)** to present key factors driving student academic success.

Perfect for showcasing clean data science workflows and interactive frontend capabilities!

---

## 🚀 Core Research Questions Answered

### 1️⃣ Does class attendance affect final marks?
* **Statistical Answer:** **Yes, extremely strongly.** 
* **Details:** We computed a strong Pearson correlation coefficient of **r = +0.81** between attendance and total marks. Students with critical attendance (**<70%**) scored an average of only **35.4%** (leading to a high failure rate). In contrast, students with excellent attendance (**>=90%**) scored a highly competitive average of **78.5%**.
* **Takeaway:** Class attendance is the single strongest factor in academic success in this dataset.

### 2️⃣ Which academic subjects carry the highest failure rates?
* **Statistical Answer:** **Mathematics** is the most challenging subject, followed by **Physics**.
* **Failure Rankings Table:**
  | Rank | Subject | Average Marks | Failure Rate (%) | Difficulty Profile |
  | :--- | :--- | :---: | :---: | :--- |
  | 1 | **Mathematics** | 49.3% | **28.3%** | 🔴 Critical / High Challenge |
  | 2 | **Physics** | 53.6% | **18.2%** | 🟡 Moderate Challenge |
  | 3 | **Chemistry** | 58.1% | **7.5%** | 🟢 Low-Moderate Challenge |
  | 4 | **Computer Science** | 60.3% | **5.0%** | 🟢 Accessible |
  | 5 | **English** | 61.4% | **0.0%** | 🟢 High Success Rate |

### 3️⃣ Does weekly study hours directly improve student scores?
* **Statistical Answer:** **Yes, with linear returns.**
* **Details:** The Pearson correlation coefficient stands at **r = +0.76**, indicating a strong positive relationship. Dividing students into study brackets highlights the performance scaling:
  * **Low Study Hours (<6 hrs/wk):** Average score is **45.2%** with a **31.2% fail rate**.
  * **Medium Study Hours (6-15 hrs/wk):** Average score rises to **58.7%** with an **8.5% fail rate**.
  * **High Study Hours (>15 hrs/wk):** Average score peaks at **74.2%** with a **0.0% failure rate (100% Pass)**.

## 💻 Setup & Local Run Guide

### Part A: Running the Python Data Pipeline
You can easily regenerate the dataset and re-execute the mathematical analysis locally using Python.

1. **Navigate to the project folder:**
   ```bash
   cd calculator
   ```
2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On macOS/Linux
   venv\Scripts\activate     # On Windows
   ```
3. **Install the required libraries:**
   ```bash
   pip install pandas numpy openpyxl matplotlib seaborn
   ```
4. **Generate the CSV/Excel files:**
   ```bash
   python generate_dataset.py
   ```
5. **Execute the statistical analysis:**
   ```bash
   python analyze_data.py
   ```
   *This outputs `analysis_summary.json` and creates static plots in the `visualizations/` folder!*

### Part B: Launching the Web Dashboard
Since the dashboard has a **CORS-safe built-in JavaScript dataset engine**, you can run it instantly using any of the following methods:

* **Method 1 (Super Simple):** Direct open. Double-click the `index.html` file in your explorer to run the page completely in your browser. (The uploader will allow parsing custom files local-only).
* **Method 2 (Visual Studio Code):** Right-click `index.html` and click **"Open with Live Server"**.
* **Method 3 (Python Local Server):** Run this command in your terminal and visit `http://localhost:8000`:
  ```bash
  python -m http.server 8000
  ```
