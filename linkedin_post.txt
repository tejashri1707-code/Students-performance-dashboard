🚀 Portfolio Showcase: Interactive Student Performance Analytics Dashboard 🎓

I am excited to share a project I've been working on that demonstrates the power of combining data engineering, quantitative analytics, and premium user-interface design.

This project is an end-to-end Student Performance Analysis pipeline. It evaluates the educational metrics of 600 student profiles to uncover key statistical trends driving academic success.

📂 GitHub Repository: https://github.com/tejashri1707-code/Students-performance-dashboard
📊 Interactive Demo: https://tejashri1707-code.github.io/Students-performance-dashboard/

🔍 Here are the core questions solved mathematically using Python (Pandas) and dynamic front-end visualizations:

1️⃣ Does class attendance affect final marks?
👉 Absolutely. We calculated a very strong Pearson correlation coefficient (r = +0.81). Students with low attendance (<70%) are highly at risk, averaging a critical score of just 35.4% (below passing). Meanwhile, excellent attendance (>=90%) drives averages to a strong 78.5%!

2️⃣ Which academic subjects yield the highest failure rates?
👉 Mathematics proved to be the most challenging subject, carrying a failure rate of 28.3%. Physics followed with 18.2%. Conversely, English was highly accessible with a 0% failure rate!

3️⃣ Does weekly study hours directly improve exam grades?
👉 Yes. A solid positive correlation of r = +0.76 is observed. The data proves a strong linear return on study hours, and students crossing the 15 hours/week benchmark achieved a flawless 100% pass rate!

💻 Technical Implementation Highlights:
🔹 Python & Pandas: Data extraction, clearing, Pearson correlation modeling, and grouping statistics.
🔹 Professional Excel Styling: Used Pandas & OpenPyXL to save the database as a cleanly structured spreadsheet showing corporate Excel modeling capability.
🔹 Front-end Analytics: Developed a responsive, premium glassmorphism dark-mode UI utilizing JavaScript ES6 and Chart.js for smooth animated canvas charts.
🔹 Local Database Uploader: Implemented a dynamic drag-and-drop CSV parser using PapaParse, allowing recruiters/visitors to upload their own classroom records and watch the entire dashboard recalculate in real-time under 50ms!

Building this project helped me sharpen my full-cycle data capabilities—from structuring raw synthetic records to conducting statistical significance analysis and building custom visual interactive experiences.
